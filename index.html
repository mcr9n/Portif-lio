Competências

Quais são as minhas principais áreas de estudo?

Durante um período de quase três anos de estudos na área de tecnologia, desenvolvi algumas competências interessantes que podem ser utilizadas no mercado de trabalho.

1

Redes de computadores

Bom conhecimento teórico em redes de computadores: algoritmos de roteamento, protocolos de comunicação, camadas de redes, programação em baixo nível.
Gostaria muito de ter a oportunidade de aplicar esses conhecimentos no mercado de trabalho.

2

Linguagens de programação

As linguagens de programação são as ferramentas que utilizamos para criar programas, aplicativos e "concretizar" algoritmos. Elas são responsáveis por fazer com que o computador execute as tarefas que desejamos.
São ótimas ferramentas para quem deseja trabalhar com desenvolvimento de software, pois permitem que você crie programas e aplicativos para diversas plataformas, como web, desktop e mobile.

3

Organização e arquitetura de computadores

Conhecimento teórico sobre o funcionamento de processadores em geral e como se dá sua construção, organização de componentes e processamento de dados.
Além disso, tenho experiência na utilização de arquiteturas de conjunto de instruções para programação de processadores em baixo nível utilizando linguagem Assembly.

4

Banco de dados

Tenho conhecimento teórico sobre o funcionamento de bancos de dados, como se dá a criação de tabelas, relacionamentos entre elas, como se dá a inserção, remoção e atualização de dados.
Ademais tenho experiência com projeto de bancos de dados, que envolvem diagrama entidade relacionamento, modelo relacional e criação de scripts utilizando SQL.
